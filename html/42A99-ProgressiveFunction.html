progressive function


Definition

A function f∈L2⁢(ℝ) is called progressive iff its Fourier transform is supported by positive frequencies only:

suppf^⊆ℝ+.

It is called regressive iff the time reversed function f⁢(-t) is progressive, or equivalently, if

suppf^⊆ℝ-.
